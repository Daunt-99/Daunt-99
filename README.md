## Featured Projects

### Parkinson's Detect
An ML system that analyzes voice biomarkers to detect early-stage Parkinson's disease with **85%+ accuracy**. Presented at AIMD Spring 2025 — placed 2nd overall.

The core insight: subtle changes in vocal frequency, jitter, and shimmer correlate strongly with neurological degradation long before other symptoms appear. Built a full pipeline from raw audio feature extraction through model training and evaluation.

`Python` `Scikit-learn` `NumPy` `Pandas`

---

### TravelBudgetFX
An AI-powered travel planning platform that automates trip budgeting and generates custom itineraries. Built to solve the gap between travel inspiration and the actual logistics of budgeting across currencies and destinations.

Integrates Gemini AI for natural language itinerary generation with a FastAPI backend and a TypeScript frontend that handles real-time FX conversion.

`FastAPI` `TypeScript` `Gemini AI` `REST API`
